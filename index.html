Find and Exchange Currency

Currency Calculator is a project made to aid those traveling to places containing different forms of currency, or those who are curious about currency exchange. The site allows users to search by country for the type of currency that is used there on the "Currency Lookup" page. Users are able to convert one currency to another on the "Currency Converter" page.

Lookup Currency

Users are able to search by country to determine the name of the currency used there. Users can type the country name into the text field and click search to display the country's currency below the search bar. This feature utilizes the REST Countries API to gather data on countries and the Currency Name page was made by Lauren Dostal.

Calculate Currency Exchange

Users are able to calculate the currency exchange between two types of currency using the Currency Converter. The user can select a currency they wish to translate, enter a number or use the arrows to select a value in the text field, then select the currency they would like the value converted to. This feature utilizes Frankfurter Exchange API and the Currency Converter page was made by Laura Ellingson.
