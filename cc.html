Currency Converter

Calculate Currency Exchange

Calculate currency exchange rates between two different currencies using the Currency Converter. Select a currency you wish to convert, enter a number or use the arrows to select a value in the text field, then select the currency you would like the value converted to. This feature utilizes Frankfurter Exchange API and the Currency Converter page was made by Laura Ellingson.
